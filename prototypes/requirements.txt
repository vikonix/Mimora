# The prototypes reuse the project's packages, so install these first:

#   pip install -r pronounce/requirements.txt   # pronounce core (Wav2Vec2, fastdtw, phonemizer-fork)
#   pip install -r prototypes/requirements.txt  # this file
#
# Native prerequisite (already required by pronounce/): the espeak-ng binary on PATH.
#   Windows : https://github.com/espeak-ng/espeak-ng/releases
#   macOS   : brew install espeak-ng
#   Linux   : sudo apt-get install espeak-ng
